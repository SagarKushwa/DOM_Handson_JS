Question:8 - What is the difference between window vs document?

Answer:-

The main difference between window and document is that window represents the browser window and provides access to browser-level functionalities, while document represents the currently loaded HTML document and provides access to the structure and content of the web page. They are closely related and often used together in web development to create dynamic and interactive web applications.
